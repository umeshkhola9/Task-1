# Internship Program of Web Development with HTML & CSS & JavaScript by ApexPlanet Software Pvt. Ltd.

Task 1 :-
A simple 4-page website built with plain HTML5 and CSS3.

## Pages

- `index.html` – Home
- `about.html` – About
- `services.html` – Services
- `contact.html` – Contact

## Folder structure

```
Task 1/
├── index.html

├── contact.html

    └── logo.png
        other images
```

## Required tags used on every page

- `<header>`, `<nav>`, `<main>`, `<footer>` — page structure.
- `<section>` — content groups.
- `<article>` — self-contained cards (services, team values, etc.).
- `<aside>` — sidebar/complementary content (quick facts, location, free-quote box).
- `<form>` — contact form (name, email, phone, message).
- `<table>` — pricing plans (Services) and office hours (Contact).
- `<ul>` / `<li>` — nav menus, skills list, footer links.
- `<img>` — logo and portfolio gallery, all with descriptive `alt` text.
